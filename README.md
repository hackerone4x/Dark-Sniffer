# Dark-Sniffer
This Is A python Sniffer To Get Packets Data And source and Destenation IP and Mac Adress And Port And Size Of Packet.
#### Installation :
- >   apt install python3 
- >   apt install git 
- >   git clone https://github.com/hackerone4x/Dark-Sniffer
- >   cd Dark-Sniffer
- >   pip3 install -r requirements.txt
- >   python3 sniffer.py
